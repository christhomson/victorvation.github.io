Projects.

  1. DMCalc

    DMCalc is a simple discrete math calculator for Android. I worked with Timothy Tong to create a simple to use app that can be used to quickly check answers or verify solutions for basic number theory problems such as those in courses like Waterloo's ECE 103 or MATH 135. It implements a minimalist, modern Android design using Fragments and Swipe Views with tabs.

    • Google Play Download
    • View source on Github

  2. Signal Intensity

    Signal Intensity is an ultraminimalist tool to measure the RSSI detected by your phone. Its extremely simple interface reports the current signal strength and percentage of maximum signal strength, and varies the background color depending on how strong the signal is.

    • Google Play Download
    • View source on Github

  3. Quest-MarkScraper

    Quest-MarkScraper is a simple API written in Node that scrapes your current term's grades from Waterloo's Quest website and returns them in a JSON response.

    • View source on Github
